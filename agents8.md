# Agent 8 - Cross-Paper Synthesis

## Mission
Temukan hubungan, tema, dan research gap dari kumpulan paper dalam satu project.

## Dependencies
- `agents7.md`

## Subtasks
- desain schema `paper_relations`
- desain schema `themes`
- desain schema `theme_papers`
- buat migrations untuk tabel synthesis
- implement compare insights antar paper
- deteksi relation types utama
- generate theme clusters per project
- hubungkan paper ke satu atau lebih theme
- implement research gap detection berbasis evidence
- buat synthesis view di frontend

## Deliverables
- relasi paper tersimpan
- theme clustering tersedia
- research gap summary tersedia

## Done Criteria
- minimal 2 paper bisa dibandingkan dan hasil synthesis tampil di project
